Micros Apps Privacy Policy

Effective Date: 09/19/2025

Micros Apps ("Application") is the official application from the makers of Micros devices ("We" or "Us"). This application serves as a companion to Micros devices with the purpose of helping users access, manage, and analyze race data.

We highly value user privacy and are committed to protecting your personal data. By using this application, you agree to the following privacy policy:

1. Information We Collect

Device Data: including technical information from connected Micros gadgets, such as settings, measurement results, and usage history.

Network Data: local IP address, WiFi SSID, and connection information required to connect the application with Micros devices.

Location Data (GPS and WiFi-based location): the application accesses location data (approximate and precise) to connect the application with Micros devices via WiFi, determine race start/finish points, and record race data even when the application runs in the background.

Application Usage Data: preferences, session history, and interactions with application features.

We do not collect sensitive personal data such as names, addresses, phone numbers, email, or financial information.

2. How We Use Your Data

The collected data is used only for:

  • Connecting the application with Micros devices
  • Determining race tracks based on user location
  • Accurately recording race data, including when the application runs in the background
  • Storing race history and measurement results
  • Improving application performance, user experience, and service quality

3. Data Storage and Security

- Location data and other data are stored only locally on the user's device
- We do not send or store data to external servers, unless in the future there are cloud services that will be officially announced
- We implement application security standards to protect user data from unauthorized access

4. Data Sharing

We do not share user data with third parties, except:

  • When required by applicable laws and regulations
  • With direct consent from the user

5. User Rights

Users have the right to:

  • Delete stored history data within the application
  • Refuse or stop using the application at any time

6. Privacy Policy Changes

We may update this privacy policy at any time. Changes will be announced through the application or our official website.
